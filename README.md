# 💰 Smart Expense Tracker 

Smart Expense Tracker is a full-stack web application that helps users efficiently manage their personal finances by tracking income and expenses in a simple and intuitive way. It provides a clean dashboard with real-time insights, allowing users to monitor their spending habits and make better financial decisions.

## 🚀 Features

* 🔐 User Authentication (Signup/Login)
* 💸 Add, edit, and delete transactions
* 🗂 Categorize income and expenses
* 📊 Interactive dashboard with charts and analytics
* 🎯 Monthly budget tracking with alerts
* 📅 Filter transactions by date and category
* 📈 Spending insights and trends
* 📤 Export data (PDF/CSV)

## 🛠 Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Authentication:** JWT

## 📊 Key Highlights

* Real-time financial tracking
* Clean and responsive UI
* Data visualization using charts
* Secure and scalable architecture

## 📌 Use Case

This application is designed for individuals who want to track their daily expenses, manage budgets, and gain insights into their financial habits.
